## Description

[Nest](https://github.com/nestjs/nest)experimentation to discover the possibilities of the framework

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run webpack
$ node dist/server.js
  
# run
$ yarn run start
  
# generate the swagger file
$ yarn run swagger

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
